---
title: 'Speko'
description: 'One API for every voice provider. Speko benchmarks STT, LLM, and TTS in real time and routes each call to the best provider for your language and latency/cost target.'
---

Speko is a voice gateway. You ship one integration; we route each request to the highest-scoring provider for your `(language, optimizeFor)` intent. Failover is server-side. Providers rotate without a code change.

## Two ways to integrate

<CardGroup cols={2}>
  <Card title="One-shot HTTP" icon="terminal" href="/guides/one-shot">
    Call `/v1/transcribe`, `/v1/synthesize`, `/v1/complete` directly. Best for batch jobs and server pipelines.
  </Card>
  <Card title="Browser conversation" icon="browser" href="/guides/realtime-conversation">
    Real-time voice in the browser. Mic in, agent voice out, transcripts on the data channel.
  </Card>
</CardGroup>
