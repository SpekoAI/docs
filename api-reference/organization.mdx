## GET /v1/organization

Returns the organization the API key belongs to.

### Response (`200 OK`)

```json
{
  "id": "...",
  "name": "Acme Health",
  "slug": "acme-health",
  "createdAt": "...",
  "metadata": null
}
```

Exact fields match the underlying `organization` row from the auth schema.

### Errors

| HTTP | When |
| --- | --- |
| `404` | Organization not found (rare; usually means stale auth context). |
