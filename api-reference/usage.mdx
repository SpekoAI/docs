## GET /v1/usage

Returns session count plus per-provider, per-metric usage for the requested window.

### Query parameters

| Param | Type | Notes |
| --- | --- | --- |
| `from` | ISO date/datetime | Inclusive lower bound. Optional; defaults to start of current period. |
| `to` | ISO date/datetime | Inclusive upper bound. Optional; defaults to now. |

### Response (`200 OK`)

```json
{
  "totalSessions": 42,
  "totalMinutes": 0,
  "totalCost": 0,
  "breakdown": [
    { "provider": "deepgram", "metric": "stt_seconds", "quantity": 12345, "cost": 0 },
    { "provider": "openai",   "metric": "llm_tokens",  "quantity": 67890, "cost": 0 },
    { "provider": "cartesia", "metric": "tts_characters", "quantity": 23456, "cost": 0 }
  ]
}
```

`totalMinutes` and per-row `cost` are placeholders (`0`) until cost rollup ships. `quantity` is authoritative; multiply by your provider rate card for cost.

`provider` is the provider that *actually ran* (post-failover), not the original top pick.
