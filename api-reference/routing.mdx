## GET /v1/routing/preview

Preview the router's choice for an intent without making a transcribe / synthesize / complete call. No usage is recorded.

### Query parameters

| Param | Type | Required | Notes |
| --- | --- | --- | --- |
| `language` | string | yes | BCP-47 (`en`, `es-MX`, …). |
| `vertical` | enum | yes | `general` \| `healthcare` \| `finance` \| `legal`. |
| `optimize_for` | enum | no | `balanced` (default) \| `accuracy` \| `latency` \| `cost`. |

### Response (`200 OK`)

```json
{
  "stt": {
    "providerId": "deepgram",
    "modelId": "nova-2",
    "score": 0.91,
    "runnersUp": [
      { "providerId": "assemblyai", "modelId": "best", "score": 0.84 }
    ]
  },
  "llm": {
    "providerId": "openai",
    "modelId": "gpt-4o-mini",
    "score": 0.88,
    "runnersUp": []
  },
  "tts": {
    "providerId": "cartesia",
    "modelId": "sonic",
    "score": 0.86,
    "runnersUp": [
      { "providerId": "elevenlabs", "modelId": "eleven_turbo_v2", "score": 0.79 }
    ]
  },
  "intent": { "language": "en-US", "vertical": "general", "optimizeFor": "balanced" },
  "scoresRunId": "..."
}
```

Each modality may be `null` if no benchmarked provider matches the intent.

### Errors

| HTTP | Code | When |
| --- | --- | --- |
| `400` | `INVALID_INTENT` | Bad query parameters. |
| `5xx` | `PROVIDER_SELECTION_FAILED` | No benchmarked candidate matches. |
